# esp-pbft

PBFT (Practical Byzantine Fault Tolerance) consensus core for ESP32 family chips.

> **Status: design only.** The `src/` and `include/` directories are skeletons.
> All design, wire format, crypto, and verification work lives in [`docs/`](./docs/INDEX.md).
> The implementation plan is in [`docs/ROADMAP.md`](./docs/ROADMAP.md) (P1–P5, ~13 weeks).
> Component manifest declares ESP-IDF `>=6.0.2`; primary target is **esp32c3**; other
> listed targets are unvalidated.
